# Product Marketing Context

_Last updated: 2026-07-20 · Constitution v2.1; prototype and external legal validation pending._

## Product overview

**One-liner:**
你不下單，你排席。五個有連續人生的小人，封席後會自己做決定。

**What it does:**
《盤勢・眾生》是一款五人 AI 命運策略遊戲。玩家每五個有效交易日編成一回合的五份公司卷宗，再把五個人與卷宗排進五張固定桌位。桌位決定風險、資訊視角與一名同伴來源；角色自行決定公司、上行／下行／不做及信心部位。揭曉時顯示人物因果與一個可驗證反事實。

**Product category:**
AI character-management strategy game with sealed historical market episodes.

**Product type:**
Consumer game. It is not a brokerage, investment adviser, stock screener, signal service, or portfolio tracker.

**Business model:**
封測完整開放、不接金流、不顯示廣告。正式歷史版預定提供延遲公共觀看的免費層，以及可排自己的五人、看完整歷史與反事實、無廣告的訂閱層。付費不提供較早資料、較準模型或績效優勢。

## Target audience

**Primary audience:**
喜歡人物養成、隊伍配置與臺股文化，願意觀察同一份封存事實如何被不同人格讀出不同決定的人。

**Jobs to be done:**

- 每回合編好五份卷宗，每天用一分鐘安排人與位置。
- 回看注意、記憶、關係與情緒如何造成可證偽的行動差異。
- 在八回合分區中學會自己的隊伍，而不是猜下一檔股票。
- 觀看退到公開世界的角色繼續生活，不消失也不被陌生人接管。

**Anti-persona:**
尋找即時個股建議、可照抄交易、保證勝率、模型能力榜或真錢報酬的人。

## Core tension

玩家看得見每次排席如何改變合法行為，卻看不見角色最後會選什麼。角色的決定要讓人感到「我能理解原因，但不能替他重選」。只有隨機就沒有養成；玩家能指定答案時，人物只剩包裝。

## Differentiation

- **排人，不排答案。** 玩家放人物與卷宗；桌位綁定資訊、風險與同伴來源。
- **自主可以驗證。** 封席後不能重抽；揭曉提供第一／第二行動、效用差與單變量反事實。
- **玄學進入人物，不進入股價。** 命盤改變注意、意義與行動門檻，不宣稱預測市場。
- **先做歷史隔離版。** 公開首版用虛構公司名和遮蔽日期；當期模式不是架構預設。
- **競賽獎勵校準。** 產業相對結果與決策分懲罰無優勢的大信心，不用全球報酬榜。
- **角色不被付款綁架。** 失訂後自主生活；恢復時取回同一五人的排席權。

## Objections

| Objection | Response |
| --- | --- |
| 「這是在幫我選股票嗎？」 | 公開首版不顯示當期 ticker、真實日期或可匯出的交易清單，也不接券商。當期模式須另過書面法遵與資料授權。 |
| 「每天只是換位置嗎？」 | 換位會同時改變風險上限、資訊視角、四家公司與唯一同伴來源；第三天還要在脈絡深度和拆散共識之間取捨。 |
| 「人格和命盤只是故事嗎？」 | 每個因素要通過單變量反事實與影響率測試；不能改變決定的特徵不進核心。 |
| 「退訂是不是把角色扣住？」 | 人物不消失、不死亡、不被別人控制。玩家在回合邊界失去或恢復排席權，人生仍持續。 |

## Customer language

**Words to use:**
排席、五席圓桌、卷宗、封席、揭曉、關鍵因果、反事實、脈絡深度、風險煞車、公開自主、思考核心、上行／下行立場。

**Words to avoid:**
AI 投顧、選股助手、神準、必漲、財富密碼、跟單、提高勝率、智商、模型戰力、贖回角色、錯過行情、即時操盤、下單、成交、融券。

**Glossary:**

| Term | Meaning |
| --- | --- |
| 排席 | 將五個人與五份週卷宗放到五張固定桌位。 |
| 卷宗 | 四家歷史 episode 公司與其封存證據；五個有效交易日固定。 |
| 封席 | 角色開始形成決定；之後不可提前讀取或改寫。 |
| 關鍵因果 | 經單變量反事實證明，確實改變最後行動的因素。 |
| 思考核心 | 版本化的證據解讀能力；不直接選公司、方向或信心。 |
| 公開自主 | 玩家失去排席權後，角色延續人生、只能被延遲觀看的狀態。 |

## Brand voice

**Tone:**
沉著、敏銳、有人的溫度，帶一點玄學的幽微感；面對 episode、分數、證據與風險時非常直白。

**Traits:**

- 神祕，但不預言。
- 有生命，但不濫情。
- 有策略，但不像券商。
- 有電影感，但不堆裝飾。
- 有觀點，但不承諾結果。

**Writing rules:**

- 先寫玩家眼前的人、桌位與變化，再寫世界觀。
- 一個畫面只用一個主要隱喻；資料狀態不用詩意替代。
- 少用口號、排比與「不是……而是……」句式。
- 公司、episode、分數、資料缺漏與風險用精確文字。
- 命盤只描述注意、衝突與象徵，不寫成行情因果。
- 「因為」只用於反事實確實改變最後選擇的情況。

## Visual language

墨黑、紙色、收斂的銅與一個冷色資料訊號。核心物件是一張五席圓桌、五個人物與五份可拖曳卷宗。命盤紋理只在封席和反事實切換時微動；所有 episode、分數、風險與資料狀態保持高對比，並提供 reduced-motion 等價畫面。避免賭場閃光、霓虹終端、裝飾性粒子與通用 AI 漸層。

## Proof before claim

| Claim theme | Evidence required |
| --- | --- |
| 小人真的自主 | 玩家不能指定最後公司、方向或信心；封席決定可法證重播。 |
| 人物真的不同 | 單變量反事實、因素影響率與角色差異率落在憲法區間。 |
| 排席真的有策略 | 第三日後 80% 測試者能說出一條位置如何改變角色決定。 |
| 不提供當期訊號 | 虛構名、遮蔽日期、無 ticker 搜尋／匯出／券商連結；當期模式另過 gate。 |
| 失訂仍有尊嚴 | 無死亡、贖回、競標、情緒勒索；回合邊界可逆且時間線不回檔。 |

## Current goals

**Product goal:**
先證明「排席 → 封席自主 → 揭曉反事實」在 60 秒內有策略感、人物感與等待張力。

**Conversion action:**
免費觀看一段已完成的角色關鍵因果後，建立自己的五人隊伍並完成第一次排席。

**Current metrics:**
尚無可用數據。首輪只驗證理解、完成時間、第三日歸因、角色差異、盲策略支配、訊號誤讀與失訂感受。
