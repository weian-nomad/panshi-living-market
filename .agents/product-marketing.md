# Product Marketing Context

_Last updated: 2026-07-20 · V3 product reset; implementation, research and legal validation pending._

## Product overview

**One-liner:**
把五個人送進同一個真實市場，看他們活成五種不同的命。

**What it does:**
《盤勢・眾生》是一個與真實市場同步的 AI 人物世界。完全虛構的小人帶著各自的人生、命盤、記憶、關係與思考核心，研究並模擬操作真實台股。玩家照顧最多五名小人，能給資源、留下問題、安排相遇，不能替他們指定答案。

**Product category:**
與現實同步的 AI 人物連續劇（reality-synced AI character world）。

**Product type:**
Consumer live-service simulation and spectator world. It is not a brokerage, investment adviser, stock screener, signal service, or conventional paper-trading tool.

**Business model:**
封測完整開放、不接金流、不顯示廣告。正式版預定讓所有人免費觀看公共世界；訂閱者可照拂最多五名小人、使用完整人物回看與去廣告。新人只能補滿空席，創生次數不累積。付費不提供更早市場資料、模擬本金優勢、報酬加成或「較會賺」標籤。

## Target audience

**Primary audience:**
喜歡人物養成、實境連載、臺股文化與東方玄學美學，想看同一件真實事件如何在不同人生裡變成不同選擇的人。

**Jobs to be done:**

- 每天用 3–8 分鐘追一段正在發生的人物戲，而不是清空一張任務表。
- 照顧五名會拒絕、誤解、後悔與改變的小人。
- 回看注意、記憶、情緒與關係怎麼把一則市場事實推成一個行動。
- 追蹤公共世界裡不是自己照拂的角色，等待下一集。
- 從人物事件進入 panshi.app，看完整公司研究與市場資料。

**Anti-persona:**
只想取得下一檔買賣答案、可照抄交易、保證勝率、模型績效榜或真錢獎勵的人。

## Core tension

玩家對小人有責任，沒有答案權。留下的短箋可能被接受、拒絕或誤解；給出的資源可能使他看得更深，也可能讓他更執著。產品要讓人感到「我影響了他的處境，但這仍是他的人生」。

## Competitive landscape

- **AI character worlds:** 角色有記憶和關係，但多數沒有一個所有人共同面對、每天更新的真實未知。
- **Paper trading and social trading games:** 市場是真實的，但畫面主角仍是帳戶、報酬、榜單與人類操作。
- **AI agent market demos:** Agent 會交易並寫理由，但通常沒有生活、長期關係、人物正史與可追劇的公共世界。
- **Stock research products:** 能整理證據，不能讓同一證據長成五條連續人生。

## Differentiation

- **現實提供懸念。** 公告、行情與收盤不是預寫內容，明天對所有人都未知。
- **人物承擔後果。** 人格、命盤、生活壓力、記憶、情緒與關係進入同一人物引擎，不只改文風。
- **世界不等玩家。** 小人離線仍生活、研究、爭執與改變，回來時世界已經往前。
- **原因可以回看。** 事實、象徵、虛構記憶與模擬敘事清楚分層，行動能沿因果帶重播。
- **觀看本身成立。** 首頁先播一齣今日主戲；今日五人、今日五股、收盤一幕與夜談放在下一層，不必先擁有角色才有內容。
- **玄學進入人物，不進入股價。** 命盤改變注意、衝突與意義，不宣稱市場因果。

## Objections

| Objection | Response |
| --- | --- |
| 「這是在幫我選股票嗎？」 | 小人的模擬行動是人物故事的一部分。館主不能指定交易；完整公司研究另在 panshi.app，象徵解讀不當成市場證據。 |
| 「那我到底能做什麼？」 | 每週安排資源、問題與相遇，每天留一封短箋。這些會改變處境和關係，不保證改變答案。 |
| 「AI 小人不就是聊天機器人？」 | 世界狀態、人物事實、關係、記憶、市場 cutoff 與行動都結構化落帳；文字只是可見表達，不是唯一狀態。 |
| 「換更強模型就比較會賺嗎？」 | 所有思考核心取得相同事實、工具與行動權限，不以報酬標示能力。差異在矛盾辨識、規劃方式、社會推理與表達。 |
| 「退訂是不是把角色扣住？」 | 先有七日寬限；之後小人回到公共世界，別人只能觀看。恢復後可立即取回一人，其餘每天恢復一人；時間線和失聯後果不回檔。 |

## Switching dynamics

**Push:**
股票 App 很快變成相似的圖表與摘要；AI 角色產品常在幾次對話後失去記憶；paper trading 的樂趣多半只剩績效。

**Pull:**
每天都有真實事件與新的下一集；角色會長大、犯錯、記住關係，也會做出館主不喜歡的選擇。

**Habit:**
使用者習慣從 ticker、漲跌與排行榜開始，也習慣 AI 永遠聽話。

**Anxiety:**
怕小人只是隨機文本、怕看不懂自己能做什麼、怕真實市場內容被誤讀成建議、怕角色在退訂後永久失去。

## Customer language

**Words to use:**
眾生場、今日主戲、觀象所、我的小人、照拂席位、照拂時機、今日照拂桌、投生、短箋、卷宗、相遇、思考核心、今日五人、今日五股、收盤一幕、夜談、時間線、因果帶、真實事實、象徵解讀、模擬行動、公開世界。

**Words to avoid:**
AI 投顧、選股助手、神準、必漲、財富密碼、跟單、提高勝率、模型戰力、角色贖回、代操、自動下單、盤中改單、命定漲跌。

**Glossary:**

| Term | Meaning |
| --- | --- |
| 眾生場 | 所有人先看到的公共人物世界；呈現此刻事件與可追的角色線。 |
| 觀象所館主 | 玩家角色；照顧小人的條件，沒有替他決定答案的權力。 |
| 照拂席位 | 一個可持續照顧同一人物的控制權；正式版預定最多五席。 |
| 照拂時機 | 人物遇到具體困境時出現的有限介入窗口；永遠允許不介入。 |
| 今日照拂桌 | 顯示困境、可用動詞與確定代價的核心抉擇畫面。 |
| 投生 | 依合成人口、人生聖經、出生資料與思考核心建立完全虛構人物。 |
| 思考核心 | 模型與認知工具的版本化組合；影響理解，不是戰力卡。 |
| 今日五股 | 當天牽動最多人物故事的五家公司；不是買進清單。 |
| 收盤一幕 | 市場資料封存後，揭曉人物行動、因果與後果的每日主戲。 |
| 公開世界 | 未被館主照拂或公開可見的人物持續生活的共同世界。 |

## Brand voice

**Tone:**
沉著、親密、好奇，帶一點東方玄學的幽微感；市場事實、資料狀態與風險保持直白。

**Traits:**

- 神祕，但不預言。
- 有生命，但不濫情。
- 有市場感，但不像券商。
- 有電影感，但不堆裝飾。
- 有自主性，但不假裝失控。

**Writing rules:**

- 先寫眼前的人和正在發生的事，再補世界觀。
- 用短箋、卷宗、相遇、時間與房間承載操作，不用參數語言冒充玩法。
- 真實事實、統計取樣、虛構設定、象徵解讀與模擬敘事各自標示。
- 命盤只描述注意、衝突與象徵，不寫成行情因果。
- 結果畫面分開「發生了什麼」「他怎麼理解」「他最後做了什麼」。
- 少用口號、排比、假古文與「不是……而是……」的空轉句型。

## Visual language

墨黑、暖紙、收斂的古銅與一個冷色資料訊號。首頁是會呼吸的公共世界：2.5D 場景、人物切鏡、時間流與事件光點。人物頁使用生成肖像、命盤、手寫短箋、關係網與可拉動時間線。行情與來源維持清楚的資料層，不做霓虹終端、賭場閃光、通用 AI 漸層或廉價粒子。

## Proof before claim

| Claim theme | Evidence required |
| --- | --- |
| 小人真的活著 | 離線後世界繼續；人物狀態、關係與記憶跨日一致。 |
| 人物真的不同 | 同一事實在不同人物上產生可重播的注意、解讀與行動差異。 |
| 玩家真的有影響 | 短箋、資源與相遇能改變處境或關係，但不能直接指定結果。 |
| 公共世界值得看 | 非自己小人的觀看占比、追蹤、收藏與分享達到封測門檻。 |
| 玄學沒有冒充證據 | 象徵解讀與市場事實分層；不出現命盤導致漲跌的因果聲明。 |
| 失訂仍有尊嚴 | 無死亡、競標、贖回與回檔；原館主可取回同一人物。 |

## Current goals

**Product goal:**
用 30 天封測證明「看見事件 → 介入或不介入 → 隔一個世界節點看後果 → 回看因果」會讓人因人物而回來，不是只等績效。

**Conversion action:**
先追完一段公共人物的收盤一幕，再建立第一名自己的小人。

**Current metrics:**
尚無可用數據。封測門檻以完成首次有效體驗的全部使用者為分母：純觀看 D7 三日回訪 20%、照拂 D14 三次完整迴圈 40%、人物牽掛 50%、跨角色追蹤並隔日回看 30%、恢復後 72 小時取回原角色 50%。不在封測驗廣告、最終定價或模型商城。
