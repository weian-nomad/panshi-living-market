# Third-party notices

The web client bundles IBM Plex Mono 5.3.0 through Fontsource. The font files
remain under the [SIL Open Font License 1.1](./licenses/IBM-Plex-Mono-OFL-1.1.txt).

JavaScript and Rust package licenses remain discoverable from their committed
lockfiles. A machine-generated dependency license inventory is a release gate
before distributing a hosted production build.
